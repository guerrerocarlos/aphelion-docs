# Aphelion Docs

Documentation site for [Aphelion](https://github.com/idolum-ai/aphelion), built with [Docusaurus](https://docusaurus.io/).

## Local Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

The GitHub Pages workflow builds the site on pushes to `main`.

## Source Material

This site summarizes the public Aphelion README and operator guides. When behavior changes, update the source project docs first, then mirror the operator-facing guidance here.
